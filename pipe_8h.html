◆ sp_pipe_create()

int sp_pipe_create ( SP_RedirOpt *  opt,
bool  nonBlocking 
)

Create a pipe at opt->value.pipeFd with the O_CLOEXEC flag.

If nonBlocking is true the O_NONBLOCK flag is also applied. If opt->type is SP_REDIR_BYTES the data is written to the pipe as soon as it is created, and the write end is closed.

Parameters
[in,out]optthe redirect option being changed.
[in]nonBlockingif true the pipe will be non-blocking.
Returns
0 on success, -1 on error and errno is set accordingly.
