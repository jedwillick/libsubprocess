int sp_redirect ( SP_RedirOpt *  opts,
SP_RedirTarget  target 
)

Redirect the target file descriptor using the given opts.

This function is intended to be used from within the child process before calling exec. If an error is encountered an error message is emmitted to stderr.

Parameters
[in,out]opts
[in]target
Returns
0 on success, or -1 on error and errno is set accordingly.
