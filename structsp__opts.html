SP_RedirTarget sp_opts::redirOrder[3]

Specifies the order to attempt redirection.

Default order is {0, 1, 2}. stdin then stdout then stderr. If given you must fully specify the unqiue order. e.g. .redirOrder = {2, 1, 0}
